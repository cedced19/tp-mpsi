## Merlin

Install `opam`

```
opam init
opam install merlin
```

Add this line to .bashrc
```
test -r /home/cjung/.opam/opam-init/init.sh && . /home/cjung/.opam/opam-init/init.sh > /dev/null 2> /dev/null || true
```
